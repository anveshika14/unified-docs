## Getting Started

To start using our API, follow these steps:

   1. **Sign Up:** Create an account on Infopass to access the API.
   2. **API Key:** Obtain your API key from the developer portal.
   3. **Documentation:** Refer to the detailed [API documentation](https://docs.infopass.in/introduction) for integration guidelines and code examples.

## API Endpoints

##### Identity Verification

 - `POST /pan/lite` : Check Pan Information
 - `POST /aadhaar/send/otp` : Send Aadhaar OTP
 - `POST /aadhaar/send/verify` : Verify Aadhaar OTP
 - `POST /aadhaar/basic` : Verify Aadhaar

##### Banking Verification

 - `POST /upi/verify` : UPI Verification
 - `POST /pennydrop/verify` : PENNY Drop Verification

##### Recharge Plan

 - `GET /operator/list` : Get Operator List
 - `POST /fetch/operator` : Fetch Operator Information
 - `POST /dth/info` : Fetch DTH Information
