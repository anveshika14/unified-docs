## Overview

Eazype provides a unified platform that combines essential services for a hassle-free experience. With their API, you can :
 - Recharge Mobile phones and DTH connections
 - Book Hotels, Flights, and Bus tickets
 - Purchase and redeem Digital Gift Cards

## Getting Started

To start using our API, follow these steps:

   1. **Sign Up:** Create an account on Eazype to access the API.
   2. **API Key:** Obtain your API key from the developer portal.
   3. **Documentation:** Refer to the detailed [API documentation](https://docs.eazype.com/introduction) for integration guidelines and code examples.

## API Endpoints

##### Mobile and DTH Recharge

 - `POST /recharge/mobile/initiate` : Initiates a mobile recharge
 - `POST /recharge/dth/initiate` : Initiates a DTH recharge
 - `GET /recharge/status` : Check the status of recharge transactions
