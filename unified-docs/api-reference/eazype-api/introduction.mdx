## Recharge APIs ✨


The Recharge API provides comprehensive DTH and mobile recharge services with resource-oriented URLs, JSON-encoded requests and responses, and standard HTTP methods and codes. It includes detailed information on offers and plans, with robust authentication for secure transactions.


| API                                                                           | Description                                     |
| :---------------------------------------------------------------------------- | :---------------------------------------------- |
| 💰 <a href="/api-reference/eazype-api/endpoints/balance">**Get Balance**</a> | Use this API to retrieve balance information  |
| 🚢 <a href="/api-reference/eazype-api/endpoints/operator-mobile">**Get Mobile Operator**</a>| Use this API to get Mobile Operator List  |
| ⛱️ <a href="/api-reference/eazype-api/endpoints/operator-dth">**Get DTH Operator**</a> | Use this API to get DTH Operator List |
| 📲 <a href="/api-reference/eazype-api/endpoints/mobile-info">**Mobile Info**</a> | Use this API to retrieving Mobile information |
| 💳 <a href="/api-reference/eazype-api/endpoints/dth-info">**DTH Info**</a> | Use this API to retrieving DTH information |
| 📠 <a href="/api-reference/eazype-api/endpoints/mobile-plan">**Mobile Plan**</a> | Use this API to get Mobile Plans |
| 🔮 <a href="/api-reference/eazype-api/endpoints/dth-plan">**DTH Plan**</a> | Use this API to get DTH Plans |
| 🏗️ <a href="/api-reference/eazype-api/endpoints/offers">**Recharge Offers**</a> | Use this API to get the offers of the recharge  |
| ⛵️ <a href="/api-reference/eazype-api/endpoints/recharge-init">**Recharge Initiate**</a>| Use this API to Initiate the recharge        |
| 💻 <a href="/api-reference/eazype-api/endpoints/check-status">**Check Status**</a>| Use this API to check the status of recharge transactions.  |
| 📝 <a href="/api-reference/eazype-api/endpoints/complaint">**Recharge Complaint**</a> | Utilize this API to raise complaints for recharge services.  |
| 📋 <a href="/api-reference/eazype-api/endpoints/complaint-status">**Complaint Status**</a> | Use this API to quickly check the status of raised recharge complaints |


## Authentication ✔

All API endpoints are authenticated using Basic auth and picked up from the specification file.

```json
"security": [
  {
    "basicAuth": []
  }
]
```
